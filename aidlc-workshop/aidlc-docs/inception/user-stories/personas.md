# 테이블오더 서비스 - 페르소나 정의

---

## 고객 페르소나 (연령대별 구분)

### Persona C1: 디지털 네이티브 고객 (20~30대)
| 항목 | 내용 |
|---|---|
| **이름** | 김민준 (28세) |
| **직업** | IT 회사 직원 |
| **디지털 숙련도** | 높음 — 스마트폰/태블릿 일상적 사용, 앱 주문 경험 풍부 |
| **목표** | 빠르고 효율적인 주문, 대기 시간 최소화 |
| **동기** | 직원 호출 없이 자유롭게 주문하고 싶음 |
| **행동 특성** | 메뉴를 빠르게 스캔, 카테고리 탐색 적극 활용, 추가 주문 빈번 |
| **불만 요소** | 느린 UI 반응, 복잡한 주문 과정, 주문 상태 확인 불가 |
| **사용 디바이스** | 매장 태블릿 (7~10인치), 개인 스마트폰 |

### Persona C2: 디지털 적응 고객 (40~60대)
| 항목 | 내용 |
|---|---|
| **이름** | 박영숙 (52세) |
| **직업** | 학원 강사 |
| **디지털 숙련도** | 중간 — 스마트폰 기본 사용 가능, 새로운 앱에 약간의 학습 필요 |
| **목표** | 쉽고 직관적인 주문, 실수 없는 주문 |
| **동기** | 가족/지인과의 식사에서 편리하게 주문하고 싶음 |
| **행동 특성** | 메뉴를 꼼꼼히 확인, 장바구니 내역 재확인, 주문 전 신중한 검토 |
| **불만 요소** | 작은 글씨/버튼, 복잡한 네비게이션, 주문 실수 시 취소 어려움 |
| **사용 디바이스** | 매장 태블릿 (7~10인치) |

## 관리자 페르소나 (역할별 구분)

### Persona A1: 점장 (Store Manager)
| 항목 | 내용 |
|---|---|
| **이름** | 이상호 (45세) |
| **역할** | 매장 총괄 관리자 |
| **디지털 숙련도** | 중상 — PC/태블릿 업무 사용 경험 있음 |
| **목표** | 매장 전체 운영 현황 파악, 메뉴/테이블 설정 관리 |
| **동기** | 효율적인 매장 운영, 인건비 절감, 고객 만족도 향상 |
| **주요 업무** | 메뉴 등록/수정, 테이블 초기 설정, 과거 주문 내역 확인, 매장 이용 완료 처리 |
| **행동 특성** | 출근 시 대시보드 확인, 메뉴 변경 시 즉시 반영, 마감 시 테이블 정리 |
| **불만 요소** | 복잡한 설정 과정, 실시간 현황 파악 어려움, 주문 누락 |
| **사용 디바이스** | 데스크톱 PC, 태블릿, 스마트폰 |

### Persona A2: 홀 매니저 (Floor Manager)
| 항목 | 내용 |
|---|---|
| **이름** | 최수진 (32세) |
| **역할** | 홀 서비스 담당, 주문 접수 및 상태 관리 |
| **디지털 숙련도** | 높음 — 모바일/태블릿 능숙 사용 |
| **목표** | 실시간 주문 모니터링, 빠른 주문 상태 변경, 테이블 회전율 관리 |
| **동기** | 원활한 홀 운영, 고객 대기 시간 최소화 |
| **주요 업무** | 실시간 주문 확인/접수, 주문 상태 변경, 테이블 이용 완료 처리, 문제 주문 삭제 |
| **행동 특성** | 대시보드 상시 모니터링, 신규 주문 즉시 확인, 테이블 상태 수시 체크 |
| **불만 요소** | 주문 알림 지연, 상태 변경 번거로움, 여러 테이블 동시 관리 어려움 |
| **사용 디바이스** | 태블릿 (이동 중), 데스크톱 (카운터) |

### Persona A3: 카운터 직원 (Counter Staff)
| 항목 | 내용 |
|---|---|
| **이름** | 정하은 (24세) |
| **역할** | 카운터 업무, 기본 주문 관리 보조 |
| **디지털 숙련도** | 높음 — 디지털 기기 능숙 |
| **목표** | 간단한 주문 확인, 테이블 이용 완료 처리 |
| **동기** | 빠르고 정확한 업무 처리 |
| **주요 업무** | 주문 현황 확인, 테이블 이용 완료 처리, 간단한 주문 상태 변경 |
| **행동 특성** | 필요 시에만 대시보드 확인, 단순 작업 위주 |
| **불만 요소** | 불필요하게 복잡한 인터페이스, 권한 과다 |
| **사용 디바이스** | 데스크톱 PC (카운터), 스마트폰 |
