## 3. 크로스 모듈 스토리 (U2 내부)

모놀리스 구조에서는 "크로스 서비스"가 아닌 "크로스 모듈"로 표현합니다. 동일 프로세스 내 함수 호출이므로 통합 복잡도가 낮습니다.

| 스토리 | 관련 모듈 | 설명 |
|---|---|---|
| US-C07 (주문 확정) | Order + Session + SSE | 주문 생성(Order) + 세션 자동 생성/귀속(Session) + 이벤트 발행(SSE) |
| US-A03 (실시간 대시보드) | Order + SSE | 데이터 API(Order) + SSE 스트림(SSE) |
| US-A08 (이용 완료) | Table/Session + SSE | 세션 종료(Session) + 이벤트 발행(SSE) |
| US-A09 (과거 주문 내역) | Order + Session | 주문 데이터(Order) + 이용 완료 시각(Session) — JOIN 쿼리 또는 SessionRepository 참조 |
